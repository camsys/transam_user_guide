dotGrants Import/Export

Admin Functionality ››

Administrators can export different types of data from TransAM into a JSON file. Likewise, administrators can upload project/funding information into TransAM using a JSON file.


Navigate to the Import/Export interface by selecting "Funding" from the main menu and then "Import/Export." From this page, administrators can either import or export a file by clicking the "New Import" or "New Export" buttons respectively.


An import simply requires you to upload the correct file from your local computer or a shared drive. After the import is complete, click on the corresponding row in the import/export table to navigate to the import detail page. The detail page will list the creator and date of the import, summarize and provide links to the import data, and communicate whether there were any errors during the import process.


When creating an export, you can select which content you would like to export:

  • Project / ALI: Whether the export should include data at the ALI or project level.
  • Fiscal year range: The fiscal year(s) included in the export.
  • Include Funding: When checked, the export will include funding data (funding requests).
  • Include Assets: When checked, the export will include asset data.


After the export is complete, click on the corresponding row in the table to navigate to the export detail page. The detail page will summarize the export information (creator, date, level, date). Click the Actions button to Download the generated file.
