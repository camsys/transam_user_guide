User Settings

Overview ››

On the right side of the banner is a user icon, which provides access to the TransAM User menu. This menu contains different options depending on your user configuration. To manage your individual settings, click this menu and select the option which lists your name. The system will redirect you to your profile page.


Every user can view and update their individual settings from their profile page. Select the Actions Menu to choose from several options:

  • Assign myself a task: Takes you to the Task page where you can create and assign tasks.
  • Update my profile picture: Select a profile picture by choosing an image from your local machine and adding the photo to your profile.
  • Update my settings: Update system settings by selecting the default number of rows you would like to display in the table views and whether you would like to receive messages in your email inbox.
  • Update my profile: Edit your personal profile to account for changes in address, title, phone number, email, etc.
  • Change my password: Change your password.


Users with management or administrative privileges have the ability perform the before mentioned actions for other users (ability to change settings and/or reset passwords for other users, etc.) as well as the ability to deactivate users within their agency.
