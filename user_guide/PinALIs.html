Pin ALIs

Capital Planning ››

TransAM allows State users to mark projects (or more specifically ALIs) as "pre-approved" to ensure these projects are programmed in an agencies capital plan. The functionality involves "pinning" an ALI to an agencies capital plan, which prevents that ALI from being further altered in any way.


State Staff and Managers can pin ALIs from all screens that display an ALI block:

  • Project Planner
  • Fund Projects (three-year view)
  • Fund Projects (single year view)
  • Capital Project Detail Page -> Activity Line Items tab


Select the orange "thumb-tack" icon to toggle the ALI between pinned (vertical thumb-tack orientation) and unpinned (horizontal thumb-tack orientation). After clicking the icon, TransAM will ask the user to confirm that they want to perform the action; once confirmed, the ALI will update.


When an ALI is marked as pinned, TransAM effectively locks that ALI in place and prevents it from being altered. This includes the following:

  • Users are not allowed to move assets from a pinned ALI to another fiscal year. Specifically, TransAM prevents users from dragging and dropping pinned ALIs to another fiscal year, and moving a sub-selection of assets from a pinned ALI to another fiscal year.
  • When applying the policy, the "Scheduled Year" for assets that are part of a pinned ALI remain unchanged.
  • When running the SOGR generator, pinned ALIs are not altered in any way. Assets attached to that ALI will remain in place; the cost of the ALI will not change; and no new assets will be added to that ALI.
  • Changes to other ALIs will not alter the "Pinned" ALI. For example, if there is a pinned ALI for FY 20-21 to replace Std 40 FT Buses, deferring two replacement Std 40 FT Buses in a FY 19-20 ALI to FY 20-21 will not add those assets to the pinned ALI. Instead, a new and separate ALI will be created in FY 20-21 for moved buses. Unpinning the ALI will not automatically merge the two ALIs; the SOGR build will need to be rerun.
  • Users are not allowed to dispose assets which are in "Pinned" ALIs. If attempted, the user will receive the pop-up warning, "This asset cannot be disposed of as it is part of a pinned ALI. Please contact your project manager for assistance." The State will need to unpin the ALI to accomplish this task.
  • The Action menu for a "Pinned" ALIs detail page is disabled
  • Pin functionality is only available for real ALIs and not shadow projects/ALIs.
