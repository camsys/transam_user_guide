The Policy information at the Agency Level includes the following:


Description

Descriptive text that describes the policy in more detail.

Condition Threshold

The condition value that indicates an asset needs to be replaced. The default value is 2.5

Depreciation Calculator

The method used to calculate the depreciated (book) value of assets. Options include Straight Line Depreciation (the default) and Declining Balance.

Depreciation Interval

The interval used to update the depreciated (book) value of assets. Options include Annual (default), Quarterly, and Monthly.
