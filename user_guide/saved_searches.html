Search/Query

Assets and Inventory ››

You can perform a query to find assets with specific attributes using the Query option on the Inventory menu. The query interface has two main tabs in the upper left corner of the page: New Query and Saved Query. The new query tab is selected by default, and with this interface you can enter search parameters based on valid combinations of Types & Subtypes, Common Fields, Physical Characteristics, and Non-Physical Characteristics, and then click the Query button. TransAM searches the database and provides a list of assets that meet your specifications.


From the list of results, you can perform the following actions:

  • Click on any asset in the list to access that asset in detail.
  • Click on the "Query Parameters" heading to expand the search parameters, where you can update and re-execute the search or reset the form to begin a new query.
  • Click the Actions Menu to "Save selection as a group" and create a Group from the results of the query.
  • Click the Actions Menu and select "Save query parameters." A form will pop up where you enter a Name and Description for the search. Saving the form will add the query to your saved queries tab.


Select the saved queries tab to view and manage your previously saved queries. From this tab, you can re-run the query, delete the query, or edit the name/description. You can even view the Structured Query Language (SQL) of the query.
