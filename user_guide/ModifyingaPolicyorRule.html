Modifying a Policy or Rule

Policy ›› Policy Actions ››

TransAM lets you make edits to a policy at all three levels:


  1. From the Actions Menu, select Modify this Policy to edit the agency-level parameters that apply to all asset types and subtypes.
  2. Click the green Alter rule button on the row describing parameters for an Asset Type.
  3. Click the green Alter rule button on the row describing parameters for an Asset Subtype.


Policy Validations

  • If you are an agency user, you are not able to change the Service Life Calculation Method.
  • If you are an agency user, you are not able to change ESL (Age)/ESL (Mileage) to values less than those set by the State.
