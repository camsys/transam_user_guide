Unconstrained Capital Projects Report

Reporting ›› Canned Reports ›› Capital Needs Reports ››

This report displays a breakdown of an agencies unconstrained capital project list.

  • If the user has multiple agencies in their organization list, the report displays a summary table of All Transit Operator projects. The table reports Organization, Project count, and Project Cost.
  • Below the summary table, the report breaks down the capital project list by agency. For each agency, TransAM lists a summary table of one capital project per row and one project attribute per column (FY, Project Number, Project Title, Scope, Cost, # ALIs, # Assets, Type, Emgcy, SOGR, Shadow, Multi Year). Click on a project to drill down and view the ALI information that makes up the project.
  • Filter the report by any combination of Fiscal Year, Emergency/Non-Emergency, Multi-Year/Single Year, or SOGR/Other.
