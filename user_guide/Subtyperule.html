Subtype rule

Policy ›› Policy Overview ››

Each Asset Type has many Subtypes. In a policy, the available Asset Subtype rules are the same as the Subtypes in your inventory. If you have a 40 FT Bus you will have a policy rule for 40 FT Buses. For rolling stock assets, each policy rule has a unique combination of asset subtype and fuel type; for non-rolling stock assets, each rule has a unique asset subtype. These rules state the useful life and replacement instructions for that distinctive subtype/fuel type combination.


The useful life information at the Asset Subtype level describes the asset and its expected lifespan. Useful life information includes:


Estimated Service Life in Months - ESL (Mo)

The number of months that assets of this subtype are expected to be used in service in a state of good repair. This might be 12 years (144 months) for a 40-foot heavy duty bus, 5 years (60 months) for a piece of IT equipment, or 30 years (360 months) for a maintenance facility.

Fuel type

The fuel type associated with the type vehicle

Estimated Service Life in Miles - ESL (Mi)

The number of miles that assets of this subtype are expected to be in service and in a state of good repair. This might be 500,000 miles for a 40-foot heavy duty bus. This item applies only to asset subtypes that track mileage.

Estimated Service Life Used in Months – ESL Used (Mo)

The number of months that assets of this subtype are expected to be in service and in a state of good repair for a used vehicle.

Lease length months

The number of months for which the asset of this subtype is scheduled to be leased.


The replacement information in the Policy dictates how TransAM will replace an asset when it reaches the end of its life. By default, TransAM assumes that at the end of its life, every asset will be replaced by an asset of the same subtype/fuel type combination. The Policy can be used to change this, so that 40-foot buses are replaced by 35-foot buses, or diesel buses by diesel-electric hybrids. Additionally, the Policy can designate if you will replace an asset with a new or leased item. The Replacement information in a Policy Rule includes:


Replacement cost

An estimate of the cost of replacing an asset of this subtype. This is usually based on estimates, existing contracts in place with vendors, or historical data. This item is ignored if the policy uses Purchase Cost + Interest as the Replacement Cost Calculator.

Cost FY year

The recorded Fiscal Year of the Replacement cost. This allows the policy to adjust the replacement cost over time due to inflation.

Replace New

An indicator that the policy is set to replace the asset with a new asset.

Replace Leased

An indicator that the policy is set to replace the asset with a leased asset.

Replace asset subtype

If assets of this subtype should be replaced by a different type of asset, this item designates the type of asset to use as the replacement.

Replace fuel type

If vehicles of this subtype should be replaced with vehicles that use a different fuel type, this item designates the fuel type to use as the replacement. This item applies only to vehicles.

Purchase replacement code

The six digit TrAMS ALI purchase replacement code as designated by the FTA (see link). This code is used to define the creation of SOGR ALIs. Administrators manage the asset subtype to purchase replacement code mappings.

Lease replacement code

The six digit TrAMS ALI lease replacement code as designated by the FTA (see link). This code is used to define the creation of SOGR ALIs. Administrators manage the asset subtype to lease replacement code mappings.

Purchase expansion code

The six digit TrAMS ALI purchase expansion code as designated by the FTA (see link). This code is used to define the creation of ALIs. Administrators manage the asset subtype to purchase expansion code mappings.

Lease expansion code

The six digit TrAMS ALI lease expansion code as designated by the FTA (see link). This code is used to define the creation of ALIs. Administrators manage the asset subtype to lease expansion code mappings.

Engineering design code

The six digit TrAMS ALI engineering design code as designated by the FTA (see link). This code is used to define the creation of ALIs. Administrators manage the asset subtype to engineering design code mappings.

Rehabilitation code

The six digit TrAMS ALI rehabilitation code as designated by the FTA (see link). This code is used to define the creation of ALIs. Administrators manage the asset subtype to rehabilitation code mappings.
