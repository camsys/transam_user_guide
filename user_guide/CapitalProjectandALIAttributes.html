Capital Project and ALI Attributes

Capital Planning ››

Every project has the following fields (as seen in the Capital Needs List):


Agency

Users with access to multiple agencies will see the agency associated with the project.

FY

The fiscal year of the project. For multi-year projects, this field takes the starting fiscal year of the project.

Project

An automatically generated project number, constructed using the Agency Name, Fiscal Year, and an automatically generated number.

Scope

The first three digits of the ALI code.

Emergency

An indicator that the project is extremely urgent.

SOGR

An indicator for whether the project is a SOGR project or a manually created project.

Shadow

Shadow projects are future predicted SOGR projects that are created by the SOGR builder.

Multi-year project

Indicates that the organization will complete this project over a span of multiple fiscal years.

Type

A designation for the type of capital project (e.g. expansion, replacement, etc.)

Title

A title assigned to the capital project. The title is automatically generated for SOGR projects.

Cost

The total expected cost of the project. For SOGR projects, this value is calculated by summing the costs of the project's ALIs.

State

The amount of state funds allocated to the project.

Federal

The amount of federal funds allocated to the project.

Local

The amount of local funds allocated to the project.

Total

The total amount of funds allocated to the project. If this number equals the project cost, then the project is fully funded. If the number is less than the project cost, then the project still requires additional funds.


The following additional fields can be viewed in the capital project's detail page:


Description

A detailed description of the capital project. For SOGR Projects, you will need to review and update this information after the builder has been run.

Justification

A detailed justification of the capital project. For SOGR Projects, you will need to review and update this information after the builder has been run.

Locations

TransAM can associate each project with one or more locations. These locations might relate to a County, UZA, Senate, Federal, or House. TransAM uses the location information associated with your agency as the default locations.

ALI List

A list of ALIs that make up the project. A capital project is the container for ALIs.


Every Activity Line Item has the following fields (as seen in a capital project detail page's Activity Line Item tab):


Name

A descriptive name assigned to the ALI item.

Fiscal Year

The fiscal year that the ALI will be completed.

Sub Category

The complete ALI code.

Count

A count of the assets that make up the ALI.

Cost

The cost of the work in the ALI estimated by the rules set in the policy. If the cost is estimated by TransAM it will be indicated by (e).

Total Funds

The total amount of funds that have been assigned to the given ALI. This field will equal the ALI cost if the ALI is fully funded.

Funds Required

The remaining funds required to fully fund the ALI. If the field is $0 the ALI is fully funded; if it is equal to the ALI cost, then the ALI has not yet been funded.

Percent Funded

The percent of the ALI that has been funded. This value is calculated based on the funded amount and the cost of the ALI.
