Editing a Project and ALI

Capital Planning ››

You can update project or ALI content using the Actions Menu on project and ALI detail pages. Use the Actions Menu to view a list of dropdown choices. SOGR and new projects have different options.


From the Actions Menu of a SOGR Capital Project, you can:


Modify this project

The Edit form of a new capital project includes the following fields: Title, Project Locations, Description, and Justification. You should update the description and justification for SOGR projects after running the SOGR Project Analyzer.


From the Actions Menu of a SOGR ALI, you can:


Update milestones

Update the list of project milestones and their corresponding estimated completion dates and comments.

Update the expected cost

The update expected cost form includes the cost estimate and a justification for the cost estimate.


From the Actions Menu of a New Capital Project, you can:


Modify this project

The Edit form of a new capital project includes the following fields: Title, Project Locations, Fiscal Year, Scope, Project Type, Emergency project, Multi-year project, Description, and Justification.

Remove this project

Remove the project and its ALIs from the capital needs list.


From the Actions Menu of a New Capital Project ALI, you can:


Update Milestones

Update the list of project milestones and their corresponding estimated completion dates and comments.

Update Expected Cost

Update expected cost includes the cost estimate and a justification for the cost estimate.

Modify this ALI

Edit form of the ALI includes the following fields - Name, Cost Estimate, Fiscal Year, Category, and Team ALI Code.

Remove this ALI

Removes the individual ALI from the project.
