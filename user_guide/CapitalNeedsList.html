Capital Needs List

Capital Planning ››

TransAM lets you perform capital planning across twelve successive fiscal years. Your capital needs list is simply a list of all the projects within this planning cycle.


You can filter the list using the Activity Line Item filter in the upper right corner of the main menu. TransAM additionally provides displays to analyze the distribution of costs and to identify funding clusters.


Like asset data, capital projects are displayed in both data tables (with one row per project, and one data item per column), and a detailed “drill down” view. When capital projects are shown in tables, you see a combination of data related to the characteristics of the project (Fiscal year, Name, Scope, etc.) and financial data (Total Cost, State, Federal, Local, etc.). Hover over a row's 'i' tooltip to see a summary of the project's ALIs.


In the detail view, project data is organized in a series of tabs. Complete the following steps to navigate from the table view to a project detail page.


1. From the Main Menu, choose Policies -> Capital Projects.

2. Click a row in the table to navigate to the detail page for the capital project.


The tabs you see in the detail view include the following:


  • Description: This tab contains a detailed description, justification, and project location information for the capital project. Edit this information from the Actions Menu.
  • Activity Line Items: This tab contains a list of the line items that compose a capital project (e.g. Purchase 5 Bus STD 35 Ft assets, Purchase 7 Bus 30 Ft assets, Purchase 3 Van asset within a single project). On a manually created capital project, you can add new line items from this tab; on a SOGR project, the line items are automatically generated and you cannot add new line items. The table includes information on the cost and allocated funds of the activity line item. Additionally, the State can select the "thumb-tack" button to lock or unlock an ALI from this tab. This is known as pinning the ALI which prohibits any changes from being made to that ALI.
  • Supporting Documents: You can attach documents to a project to ensure it is fully documented. To add a document, click the Add Document button and select your file, enter a description, and click Add Document.
  • Comments: Any user can add comments about a project to remark on its contents.


Likewise, in the detail view, ALI data is organized in a series of tabs. Complete the following steps to navigate to an ALI detail page:


  1. From the Capital Project detail page, choose the Activity Line Items tab.
  2. Click on a row in the table to navigate to the detail page for that ALI.


The tabs you see in the detail view include the following:

  • Assets: This tab contains a list of the assets associated with the ALI.
  • Cost Justification: This tab contains the explanation you provided when you updated the expected cost.
  • Milestones: A list of project milestones and their corresponding estimated completion dates and comments. Includes Out for Bid, Contract Awarded, Notice to Proceed, First Vehicle Delivered, All Vehicles Delivered, and Contract Completed.
  • Funding: A summary of the current funding that has been assigned to the ALI. The tab will list each fund that has been tagged to the ALI broken down by federal, state, and local sources.
  • Supporting Documents: You can attach documents to an ALI to ensure it is fully documented. To add a document, click the Add Document button, enter a description, and click Add Document.
  • Comments: Any user can add comments about a project to remark on its contents.
